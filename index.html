Hangman with Google Maps API Beta

The theme of this game is Countries. Can you guess the correct country?

Instructions

In this demo you will have 10 attempts to guess the randomly selected country with the animated letters provided under the instructions. *For this prototype, only 10 countires will be used which are the following: (Spain, Portugal, Peru, Switzerland, Belgium, France, Croatia, Japan, Taiwan, Brazil, Paraguay, Columbia) .The map provided below will start in the United States. If the country is guessed correctly, the country will be displayed on the map provided below. If the country was not correctly guessed, then you lose!

Click either the exit (x) button on the top right of this modal, or click the "Close" button to see the country that you attempted to guess. Or Press the "Play Again" button to try again
