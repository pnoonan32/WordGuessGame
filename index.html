Hangman with Google Maps API Beta

The theme of this game is Countries. Can you guess the correct country?

Instructions

In this demo you will have 10 attempts to guess the randomly selected country with the animated letters provided located under the instructions. For each lettter guessed wrong, a hangman piece will appear on the screen. For each letter guessed correctly, the underscores located under the empty black box with a dashed border, will be replace with the letter as they appear in the word. *For this prototype, only 10 countires will be used which are the following: (Spain, Portugal, Peru, Switzerland, Belgium, France, Croatia, Japan, Taiwan, Brazil, Paraguay, Columbia). You can track your progress throughout the game with the provided statsitics above the instructions heading. If the country is guessed correctly, the country will be displayed on the map provided below along with it's geolocation.Also, If the country was not correctly guessed, then you lose!

*Please note the country will only show up on the map provided if you guess the country correctly!

Close the modal to see the country you attempted to guess. Press the "Play Again" button located underneath the map to try again.

Location of Country



Please feel free to type in any address, state, or country you wish with the provided search bar!
